Understanding the Box Model

This page demonstrates how to connect an external CSS file and apply various styles while working with the CSS Box Model.

Margin

Margin creates space around elements, outside of any defined borders. It clears an area outside the border.

Padding

Padding creates space around content, inside of any defined borders. It clears an area around the content.

Border

The border property allows you to specify the style, width, and color of an element's border.
